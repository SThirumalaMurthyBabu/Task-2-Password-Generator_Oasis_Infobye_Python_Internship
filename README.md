# Random Password Generator using Python

This project is developed as Task 2 of the Oasis Infobyte Python Internship.

## Description
A command-line password generator that creates random passwords based on user-defined criteria such as length and character types (letters, numbers, symbols).

## Features
- User-defined password length
- Option to include letters, numbers, and symbols
- Random password generation
- Input validation

## How to Run
random_password_generator.py
